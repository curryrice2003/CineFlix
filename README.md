# CineFlix
CineFlix is a high-performance movie discovery dashboard. It provides a centralized hub for users to explore films & analyze global streaming availability, and navigate directly to official streaming platforms.

# Key Features
- Global Streaming Intelligence: Automatically fetches real-time provider data (Netflix, Disney+, Prime Video, etc.) across various global regions.

- Direct Search Routing: Bypasses intermediate landing pages to link users directly to the official search results on their selected streaming platform.

- Interactive Analytics: Features a custom-built Chart.js doughnut chart that visualizes provider distribution for any selected film.

- Subscription Filtering: Users can toggle specific services (Hulu, HBO Max, Paramount+, etc.) to find content matching their active subscriptions.

#Technical Stack
- Frontend: HTML5, Tailwind CSS

- Logic: Vanilla JavaScript (ES6+)

- Data Source: TMDB (The Movie Database) API

- Data Visualization: Chart.js

---
Note: For demonstration purposes, the API key is temporarily hardcoded. In a production environment, this would be handled via environment variables and a proxy server.
